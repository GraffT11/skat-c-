# skat in c++

## Aufgabe:

Implementiert werden soll ein vereinfachtes Skatspiel, das
• Karten mischt und die Karten der drei Spieler anzeigt,
• Den Spieler ausgibt, der die nächste Karte abwerfen muss,
• Den Spieler zur Angabe der Karte auf der Hand (Nummer der Karte) auffordert,
• Prüft, ob die abzuwerfende Karte korrekt abgeworfen wurde, ¨
• Herausfindet, wer den Stich gemacht hat,
• Die neue Runde startet oder den Gewinner und die Punktzahlen der Spieler ausgibt.
